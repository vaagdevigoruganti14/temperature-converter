# Temperature Converter

A simple web application that converts temperature values between Celsius, Fahrenheit, and Kelvin using JavaScript.

## Overview

The Temperature Converter allows users to instantly convert temperature measurements between different units through an easy-to-use interface. The project demonstrates the practical use of JavaScript for handling user input, performing calculations, and updating web content dynamically.

## Features

* Celsius to Fahrenheit conversion
* Fahrenheit to Celsius conversion
* Celsius to Kelvin conversion
* Kelvin to Celsius conversion
* Real-time calculation
* User-friendly interface
* Responsive design

## Technologies Used

* HTML5
* CSS3
* JavaScript

## Learning Outcomes

* JavaScript Functions
* DOM Manipulation
* Event Handling
* User Input Validation
* Front-End Web Development
